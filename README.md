## 💡 프로젝트 소개
선용품 카테고리 분류 및 리드타임 예측 서비스

## 🕰️ 개발 기간
* 24.08.23. - 24.10.08.

## ⚙️ 개발 환경
- `Java 17.0.10`
- `JDK 17.0.10 `
- **React** : 18.2.0
- **IDE** : STS4
- **Framework** : Springboot(2.x)
- **Database** : MySQL(8.0.37)
